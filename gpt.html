About This Menu

In this section, we will explain the process of creating the responsive menu and how it functions. Hereâ€™s a breakdown of the important parts:

1. HTML Structure

The HTML structure consists of a navigation bar


    <ul class="menu">
      <li><a href="#home">Home</a></li>
      <li><a href="#services">Services</a>
        <ul class="submenu">
          <li><a href="#web">Web Design</a></li>
        </ul>
      </li>
    </ul>

2. CSS for Styling

We use CSS to style the menu and make it responsive. Key points include:

  • Using `display: flex` for desktop layout and `display: none` for mobile by default.
  • Using `:hover` to show submenus when the user hovers over a main menu item.
  • Media queries to change the layout on smaller screens, such as displaying the menu as a vertical list and allowing it to be toggled on mobile.

3. JavaScript for Mobile Toggle

For mobile devices, we use JavaScript to allow users to toggle the menu. This is done by adding a click event to the "Menu" button (`.menu-toggle`) that toggles the visibility of the menu:


    document.querySelector('.menu-toggle').addEventListener('click', function() {
      document.querySelector('.menu').classList.toggle('active');
    });

4. Submenu Interaction

The submenus are hidden by default, and only show up when you hover over their parent menu item on desktop devices. This is achieved using the CSS rule:


    .menu li:hover > .submenu {
      display: block;
    }

On mobile, submenus will be shown when clicking their parent menu item, making it more user-friendly on smaller screens.
